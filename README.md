# 🚀 Smart Task Management API

A structured and scalable **Task Management RESTful API** built with Laravel.  
This is not just a simple To-Do app — it's a role-based productivity system designed to simulate real-world SaaS platforms.

> ⚠️ Project Status: Under Active Development

---

## 🎯 Goals

- Build a clean RESTful API using Laravel best practices  
- Implement Role-Based Access Control (RBAC)  
- Apply Policies & Authorization  
- Secure authentication with Laravel Sanctum  
- Implement database & email notifications  
- Deliver clean, maintainable, production-ready code  

---

## 🔐 Authentication

- Token-based authentication using Laravel Sanctum  
- Protected API routes  
- Role & permission-based authorization  

---

## 👥 Roles

### 🟢 Super Admin
- Full system control  
- Manage admins & users  
- Access all projects & tasks  

### 🔵 Admin
- Create users  
- Assign users to projects  
- Create & assign tasks  
- Monitor team progress  

### 🟣 User
- View assigned projects  
- Update task status  
- Receive notifications  

---

## 🗂 Core Features

- Projects & Task Management  
- Task Status (`pending`, `in_progress`, `done`)  
- Priority & Due Dates  
- Advanced Filtering & Search  
- Pagination  
- Database Notifications  
- Email Notifications  
- Postman API Documentation
